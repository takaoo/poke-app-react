# Pokemon Pokedex

## アプリの紹介

このアプリケーションは、Next.js、Tailwind CSS、そしてPokeAPIを利用して開発された高機能なポケモン図鑑（Pokedex）アプリです。ユーザーが様々な条件でポケモンを探しやすいように設計されています。

**主な機能:**
- **世代別フィルタリング**: 第1世代（Gen 1）から第9世代（Gen 9）までのポケモンを簡単に切り替えて表示できます。
- **リアルタイム検索**: 名前などによるリアルタイムなポケモンの検索が可能です。
- **タイプ別フィルタリング**: ポケモンのタイプ（ほのお、みず、くさ など）に基づいた絞り込み機能を提供します。
- **レスポンシブデザイン**: スマートフォンからデスクトップまで、様々なデバイスで快適に閲覧できるモダンなUIを採用しています。

## アプリの構成

本プロジェクトは以下のモダンな技術スタックをベースに構築されています。

- **フロントエンドフレームワーク**: Next.js (App Router)
- **スタイリング**: Tailwind CSS
- **言語**: TypeScript
- **外部API**: [PokeAPI](https://pokeapi.co/) (ポケモンの詳細データを取得)

### ディレクトリ構成の概要

- `app/`: アプリケーションのルーティング、ページコンポーネント、およびレイアウト定義
- `components/`: ポケモンカード、検索バー、フィルターなど、UIを構築する再利用可能なコンポーネント群
- `lib/`: PokeAPIからのデータ取得（フェッチング）ロジックやユーティリティ関数
- `types/`: データ構造を安全に保つためのTypeScriptの型定義
- `public/`: 画像などの静的アセット

## 開発環境のセットアップ (Getting Started)

開発サーバーを立ち上げるには、以下のコマンドを実行してください。

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) にアクセスすると、アプリケーションを確認できます。
ページの編集は `app/page.tsx` から開始でき、ファイルを保存すると自動で画面が更新されます。
